# Лабораторная работа №1: Работа с Git и GitHub

# Вариант задания: 1.c - Трапеция

# Описание программы
Программа вычисляет основные параметры трапеции:
- Периметр (сумма всех сторон)
- Площадь (полусумма оснований на высоту)
- Длину средней линии (полусумма оснований)

# Как использовать
1. Запустить программу
2. Ввести значения:
   - a - верхнее основание
   - b - нижнее основание
   - c - левая боковая сторона
   - d - правая боковая сторона
   - h - высота
3. Получить результаты вычислений
